# 🌍 WorldWise

## 📌 About the Project
WorldWise is a map tracker application built with React.  
The main goal of this project is to practice and understand **React Router** deeply by managing different pages and navigation between visited places.

Users can add cities they have visited and see them on a map interface.

---

## 🚀 Features
- Add visited cities
- View cities list
- Show city details page
- Interactive map integration
- Navigation using React Router (nested routes, dynamic routes)
- Persistent UI layout with sidebar and header

---

## 🛠️ Built With
- React
- React Router
- JavaScript (ES6+)
- CSS / Responsive Design
- API (if used in project)
